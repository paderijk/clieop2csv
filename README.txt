Information of how to use, please read the message which popup after the command:

perl clieop2psv.pl -h show
ClieOp03 to pipe-seperated-values version: 0.0.2

Options:
        -r [filename]   Define the ClieOp file to read (required)
        -s [filename]   Define the output file (required)
        -h show         Show this message

EXAMPLE:
        perl clieop2psv.pl -r c:\somepath\clieop.txt -s c:\someotherpath\clieop.psv

        -or-

        clieop2psv.pl -r /somepath/clieop.txt -s /someotherpath/clieop.psv
